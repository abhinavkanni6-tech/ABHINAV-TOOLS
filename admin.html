Admin — ABHINAV TOOLS AND BOTS

Secure admin uploader (uses a GitHub Personal Access Token). Public dashboard is separate.

Authenticate with GitHub (PAT)

To upload files from this page, paste a GitHub Personal Access Token (PAT) with public_repo scope (for a public repo) or repo (for private).

Security: The PAT stays in your browser session (sessionStorage) and is never sent to any server. Revoke the token after use if you wish.

Upload a file

Select a file, give it a display name and a short description (Markdown is supported).








Notes

  • This uploader will: 1) upload the file to uploads/<filename> in your repo; 2) update files.json to include a manifest entry so the dashboard shows the file.
  • If files.json does not exist, the uploader will create it.
  • The PAT is used only from your browser and is stored in sessionStorage; you can clear it with the "Clear token" button. Revoke the token in GitHub after use if desired.

Open repo on GitHub Back to dashboard

Important: If you previously added admin-config.json with a password, remove it from the repo to avoid exposing credentials publicly. You can delete it via the GitHub web UI (open the file → trash icon → commit).
