How to upload files (manual) and add them to the dashboard

Manual upload (recommended)

  1. Go to your repository: https://github.com/abhinavkanni6-tech/ABHINAV-TOOLS
  2. Click Add file → Upload files. Upload your file into the uploads/ folder.
  3. After upload, click Commit changes.
  4. Edit files.json (click it in the repo) and add an entry for your file. See the example below.
  5. Save the updated files.json. The dashboard will automatically show the new file.

files.json example entry

{
  "id": "example-tool-1",
  "filename": "uploads/example-tool.zip",
  "displayName": "Example Tool Pack",
  "readme": "Short description for the tool. You can use Markdown and basic HTML.",
  "url": "https://raw.githubusercontent.com/abhinavkanni6-tech/ABHINAV-TOOLS/main/uploads/example-tool.zip",
  "uploaded_at": "2026-07-17T12:00:00Z",
  "uploader": "admin"
}

Enable GitHub Pages & custom domain

  1. Go to Settings → Pages. Choose branch main and folder / (root), save.
  2. Add the DNS records at your domain registrar (see README.md for exact IPs) and keep the CNAME file in the repo.

Notes

  • Anyone can view and download files from the dashboard. Only repo admins can upload files via GitHub.
  • If you prefer scripted or token-based uploads, we can add an optional client-side uploader that uses a temporary Personal Access Token — contact me to enable.
